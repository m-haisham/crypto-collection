Hamming code is a set of error-correction codes that are used to detect and correct errors that may occur during the transmission of the data from the sender to the receiver.

The technique was developed by R.W. Hamming.

How it works

Step1: The first step is to calculate the no.of redundant bits in a message

  1. For example, if a message contains ‘n’ no.of bits and ‘p’ no.of redundant bits are added to the message, then ‘np’ indicates (n+p+1) different states.
  2. Where (n+p) represents the location of an error in every bit position
  3. 1 (extra state) represents no error.
  4. Since ‘p’ indicates 2^p (2p ) states, which are equal to (n+p+1) states.

Step2: Place the redundant bits in exact/correct position

  1. ‘p’ bits are inserted in the bit positions which are the power of 2 like 1, 2, 4, 8, 16, etc. These bit positions are indicated as p1 (position 1), p2 (position 2), p3 (position 4), etc.

Step 3: Calculate the values of redundant bits

  1. Here parity bits are used to calculate the values of redundant bits.
  2. Parity bits can make the no.of 1’s in a message either even or odd.
  3. If total no.of 1’s in a message is even, then even parity is used
  4. If total no.of 1’s in a message is odd, then odd parity is used.

Step1: Count the no.of redundant bits

The formula to encode the message using redundant bits is,

2p≥ n + p + 1

Step 2: correct the positions of all redundant bits

‘p’ no.of redundant bits are placed in a bit positions of power of 2 like 1,2,4,8,16,32 etc

Step3: parity checking (odd parity and even parity)

Parity bits are calculated based on the no.of 1’s in data bits and redundant bits.
